- 반드시 "" 사용
- [] 대괄호안에 쓸 것
- {} 객체
- 주석 사용 안됨
- "event", "web", "app", "etc"..키 값


분류해서 작업하고 싶을 때
{
    "event":[
        {"web" : [
            "../img/img-01.jpg",
            "../img/img-02.jpg",
            "../img/img-03.jpg"  <<
        ]},
        {"app" : "모바일앱 입니다.", "appName" : "자동차" },

        {"etc" : [
            "../img/img-01.jpg",
            "../img/img-02.jpg",
            "../img/img-03.jpg"  <<
        ]} << 마지막에는 ,을 지워줌
          
    ]
}
