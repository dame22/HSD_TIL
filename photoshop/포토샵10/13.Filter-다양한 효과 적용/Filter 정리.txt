
<Photoshop cs6 Filter>




<Artistic> 

color pencil : 색연필로 그린 듯한 효과를 주는 필터입니다. 배경색상이 색연필의 색상이 되기 때문에 미리 설정하신 다음 작업하시는것이 좋습니다.
 
cutout : 이미지의 색상을 단순화하여 색상톤을 단순하게 바꾸어 줍니다.
 
dry brush : 물을 적게 묻혀서 그린 것과 같은 효과를 줍니다.
 
film grain : 이미지의 색상의 밝기에 따라 차별을 주어 노이즈를 뿌려줍니다.
 
fresco : 프레스코 화풍의 효과를 내주는 필터입니다.
 
neon glow : 이미지에 네온효과를 주는 필터입니다.전경색상과 배경색상, glow color에서 선택한 색을 사용하여 네온효과를 적용합니다.
 
paint daubs : 거칠게 칠하는 붓터치의 효과를 주는 필터입니다.
 
palette knift : 유화에서 사용되는 팔레트 나이프를 가지고 그린 것과 같은 효과를 주는 필터입니다.
 
plastic wrap : 이미지에 랩을 씌운 듯한 효과를 주는 필터입니다.
 
poster edges : 이미지의 경계면에 검은 색상을 추가하여 포스터나 목판화같은 효과를 주는 필터입니다.
 
rough pastels : 파스텔그림과 같은 효과를 주는 필터입니다.회화적인 느낌이 강한 필터로 터치의 길이와 방향, 거칠기와 질감 등을 자유롭게 조절할 수 있습니다.
 
smudge stick : 그림을 물감으로 그리고 마르기 전에 문지른 것과 같은 또는 파스텔을 젖은 종이위에 그린 것과 같은 효과를 내는 필터입니다.
 
sponge : 이미지를 젖은 스폰지로 문지르는 효과를 내는 필터입니다.
 
underpaint : 색이 번진 러프 파스텔과 같은 효과를 주는 필터입니다.
 
water color : 수채화의 효과를 표현하는 필터입니다.수채화의 색 번짐이 잘 표현되지만 이미지가 어두워지기 쉽습니다.
 
 
 
<Blur> 

blur : 이미지를 흐리게 하여 부드러운 분위기를 연출하는 필터입니다.
 
blur more : blur필터를 3-4회 실행한 것과 같은 효과를 주는 필터입니다.
 
gaussian blur : 이미지에 가우스곡선이라고 불리는 수학공식을 이용하여 blur를 적용합니다. blur나 blur more보다 강력한 효과를 볼 수 있는 필터입니다.
 
motion blur : 이미지를 빠르게 움직였을 때의 잔상효과를 표현하는 필터입니다.각도와 이동거리를 조절할 수 있으며 역동적인 표현을 하고자 할 때 쓰여지는 필터입니다.
 
radial blur : 이미지를 빠르게 회전시키거나  zoom-out시킨 것과 같은 효과를 주는 필터입니다.
 
smart blur : 이미지의 색상의 경계부분을 보호하며 blur를 적용합니다.
 
 
 
<Brush stroke> 

accents edges : 경계부분을 부드럽게 처리하거나 경계부분을 두드러지게 하는 필터입니다.윤곽을 덧그리는 듯한 효과를 줍니다. 이 필터를 사용하시면 이미지가 전체적으로 밝아지게 됩니다.
 
angled strokes : 빗살을 그리는 붓터치의 효과를 내는 필터입니다.
 
crosshatch : 이미지의 대각선의 빗살무늬의 질감을 적용하는 필터입니다.
 
dark strokes : 어두움과 밝음을 조절하여 강한 빛을 쏘이는 밝은 이미지의 효과를 낼 수 있는 필터입니다.
 
ink outlines : 색상의 경계부분에 어두운 선을 그어줌으로 펜 터치를 적용하는 필터입니다.
 
spatter : 스프레이로 물감을 튀기는 효과를 연출하는 필터입니다.
 
sprayed strokes : 스프레이로 흩뿌리면서 작업한 듯 이미지를 미세하게 찌그러 뜨리는 붓터치의 효과를 적용하는 필터입니다.
 
sumi-e : 일본의 전통 수묵화 기법을 흉내내는 필터로 목탄으로 작업한것과 같은 효과를 줍니다.
 
 
 
<Distort> 

diffuse glow : 이미지의 일부분(밝은 부분)을 발광하는 효과를 줍니다.
 
displace : displacement map에 의한 이미지에 따라서 원래의 이미지를 변형시킵니다.위의 이미지는 별모양의 displacement map에 적용된 이미지입니다.
 
glass : 유리를 통해서 보는 것과 같은 효과를 낼 수 있는 필터입니다.
 
ocean ripple : 물결치는 해면에 이미지가 비추어진 듯한 효과를 주는 필터입니다.
 
pinch : 이미지를 오목거울이나.볼록거울로 본 듯한 느낌을 만드는 필터입니다.
 
polor coordinates : 극좌표의 형태로 이미지를 왜곡시킵니다.평면의 이미지를 구형으로 감싸거나 구형의 이미지를 평면에 펼치는 왜곡효과를 적용합니다.
 
ripple : 출렁이는 물 속에 넣은 이미지를 보는 것과 같은 효과를 줄 수 있습니다.
 
shear : 이미지를 직선이나 곡선의 형태로 변형시킵니다.
 
spherize : 이미지를 구형이나 원통형으로 변화시킵니다.
 
twirl : 이미지의 중심을 축으로 하여 돌리는 것처럼 회전시킵니다.
 
wave : 이미지위의 복잡한 물결모양을 만들어 주므로 다채로운 이미지를 만들 수 있습니다.
 
zigzag : 이미지에 파동의 수와 파문이 범위를 조절하여 물결의 이미지를 표현합니다.
 
 
 
<Noise> 

add noise : 이미지에 노이즈를 뿌려줍니다.monochromatic을 체크하면 검은 색상의 노이즈가 첨가 됩니다.
 
despeckle : 이미지의 노이즈를 자동으로 제거시켜 주는 필터입니다.
 
dust & scratch : 이미지의 잡티, 노이즈와 얼룩 등을 옵션의 조절로 효과적으로 제거할 수 있는 필터입니다.
 
median : radius의 조절로 노이즈를 제거합니다.수치를 높일수록 이미지가 뭉개지며, 수치가 낮을수록 미세한 변화를 줍니다.
 
 
 
   
 
<Pixelate> 

color halftone : 픽셀들을 분해하여 인쇄물을 확대하여 본 듯한 효과를 표현하는 필터입니다.망점으로 표시됩니다.
 
crystallize : 이미지를 단색의 다각형으로 분해하여 크리스탈을 통하여 보는 것과 같은 효과를 줍니다.
 
facet : 이미지의 색상을 단순화 시켜주는 필터입니다.이미지의 경계를 둥글고 부드럽게 처리해 줍니다.
 
fragment : 카메라가 좌우로 흔들리는 듯한 영상효과를 적용하는 필터입니다.
 
mezzotint : 이미지가 동판화 효과를 적용하는 필터이며 여러 가지 재질을 사용하셔서 표현이 가능합니다.
 
mosaic : 이미지에 모자이크의 효과를 적용합니다.이미지를 정사각형으로 분해하며 사이즈를 조절하실 수 있습니다.
 
pointillize : 점묘화로 그린 것과 같은 방식으로 이미지를 표현합니다.
 
 
 
 
<Render> 

3d transform : 이미지를 3D로 변형하는 필터입니다. 육각형과 원기둥형 원뿔형으로 입체감을 표현하여 줍니다.
 
clouds : 색상모드의 전경색상과 배경색상을 섞어 줍니다.구름의 이미지를 만듭니다.
 
difference clouds : clouds의 필터를 difference모드로 적용한 결과를 얻을 수 있습니다.
 
lens flare : 사진기로 찍었을 때 종종 사진에 태양빛이나 다른 광원 때문에 생기는 반사광이 생기게 되는데, 이러한 것을 만들어 내는 필터입니다. 이미지에 집중광원을 드리울 수 있습니다.
 
lighting effect : 인위적인 조명효과를 주는 필터입니다. 빛의 세기,크기,방향을 조절하여 여러가지 효과를 줄 수 있습니다.
 
 
 
<Sharpen> 

sharpen : 이미지의 경계선을 강조하여 이미지를 선명하게 해 줍니다.
 
sharpen edges : 이미지의 경계선에만 sharpen의 효과를 적용하여 이미지를 보다 선명하게 보정하여 줍니다.
 
sharpen more : 이미지에 sharpen효과를 두 번 준 것과 동일한 효과를 줍니다.
 
unsharpen mask : sharpen edges와 비슷한 효과를 보이지만, 옵션에 의해서 세밀하게 조절할 수 있습니다.
 
 
 
<sketch> 

bas relife : 이미지의 명암에 의해 입체적인 효과를 주는 필터입니다. smoothness의 수치를 낮추어 주시면 까칠까칠한 입체효과를 얻으실 수 있습니다.어두운 부분에는 전경색상을, 밝은 부분에는 배경색상이 적용됩니다.
 
chalk & charcoal : 분필과 목탄등을 사용하여 그린 이미지로 변환합니다. 쉐도우에는 전경색이 하이라이트에는 배경색이 적용됩니다. 부드러운 스케치의 효과를 주는 필터입니다.
 
chrome : 이미지의 밝은 부분과 어두운 부분을 이용하여 금속과 같은 느낌의 효과를 줍니다.이 필터를 사용하시면 이미지가 흑백으로 바뀌게 됩니다.
 
charcoal : 목탄으로 이미지의 어둡고 강렬한 부분의 특징을 잡아서 그려줍니다.
 
conte crayon : 이미지를 전경색상과 배경색상을 이용하여 크레용으로 그린 것 과 같은 효과를 적용합니다. 전경색상은 어두운 이미지를, 배경색상은 밝은 이미지를 칠하는 데 사용됩니다.
 
graphic pen : 이미지에 전경색상과 배경색상을 이용하여 명암을 표현해 주고 가는 펜으로 그린 것과 같은 효과를 줍니다.
 
halftone pattern : 이미지를 여러가지 모양의 망을 통해서 보는 것처럼 변환합니다.
 
망점의 크기와 Pattern Type을 통해 망의 종류(Cycle, Dot, LIne)도 선택할 수 있습니다.
 
note paper : 표면이 거친 종이 위에 다른 색의 종이를 겹쳐서 만든 형상으로 이미지를 변환합니다.이미지의 명도대비로 표현해 주는 필터입니다.
 
photo copy : 이미지를 복사한 것과 같은 효과를 줍니다.전경색상은 이미지의 테두리부분을, 그리고 배경부분은 배경색상으로 채워집니다.
 
plaster : plaster라는 말은 석고를 의미하는 말로,회반죽으로 이미지를 구성하는 입체적인 효과를 주는 필터입니다.금속성의 느낌이 강한 필터입니다.
 
reticulation : 이미지에 전경색상과 배경색상을 사용하여 둥근점을 조밀하게 찍어주면서 이미지를 단순화 시킵니다.
 
stamp : 이미지를 고무도장으로 찍은 것과 같이 변환됩니다.
 
torn edges : stamp와 비슷하지만 경계부분이 종이가 찢겨나간 듯 거칠게 표현됩니다.
 
water paper : 젖은 종이 위에 그림을 그린 것과 같이 변환됩니다.
 
 
 
   
 
<Stylize> 

diffuse : 이미지의 픽셀들을 흩 뿌리는 효과를 얻을 수 있습니다.
 
emboss : 이미지의 색상 경계를 통해 밝은 부분을 돌출시키는 입체 효과를 만듭니다. 주로 3D적인 입체효과를 위한 중간단계에 활용됩니다.
 
extrude : 이미지를 여러 조각으로 쪼개어 돌출시키며 조각의 크기와 모양, 돌출 정도를 조절할 수 있습니다. 쪼개어지는 형태는 블록형과 피라미드형이 있습니다.
 
find edges : 색상경계 영역을 부각시키고 다른 부분은 단순화시켜 경계선을 강조 할 수 있는 필터입니다.
 
glowing edges : 명도대비를 이용해 형광 선을 긋고 다른 부분은 어둡게 처리하여 형태를 강조 할 수 있습니다.
 
solarize : 어두운 부분의 밝기를 높여 노출과다 효과를 줍니다.
 
tiles : 이미지를 사각 타일조각으로 나누어서 흩어 놓습니다. 빈 공간의 색채는 옵션에서 지정하는 색으로 채워집니다.
 
trace contour : 명도가 바뀌는 부분에 가는 선을 그리고 나머지는 흰색을 많이 주어 윤곽선이 드러나게 합니다.
 
wind : 바람에 날린 듯한 이미지로 변환합니다. 옵션에서는 Wind, Blast, Stagger로 바람의 강도를 조정할 수 있습니다.
 
   
 
 
 
<Texture> 

craquelure : 진흙으로 벽면을 바르고 말린 듯한 효과,이미지에 틈이 있어 거칠게 느껴지는 듯한 벽화의 효과를 줍니다.
 
grain : 이미지에 노이즈를 뿌려줍니다.
 
mosaic tiles : 이미지를 타일모양으로 잘라 표현합니다.
 
patchwork : 어두운 부분의 이미지는 함몰되고, 밝은 부분의 이미지는 돌출된 사각형의 벽돌모양으로 표현해 주는 필터입니다.
 
stained glass : 스테인드글라스 모양으로 색상 영역을 잘라서 표현합니다.
 
texturizer :  이미지에 texture의 질감을 표현합니다.
 
 
 
<Video> 

de-interlace : 동영상의 이미지를 캡쳐했을 경우 이미지보정을 할 때 쓰입니다.
 
ntsc color : NTSC colors는 TV나 비디오등에서 사용되는 색을 말합니다. RGB의 이미지들을 NTSC colors로 바꾸어 줍니다.
 
 
 
<Other> 

custom : 픽셀들의 이동과 배합을 직접 조절하는 필터입니다.수치를 직접 입력하여 연산을 거쳐 이미지에 효과를 적용합니다.
 
dither box : 패턴화시킨 것을 이미지에 넣을 수 있습니다.
 
high pass : 이미지의 밝은 부분은 강조하고 그 외의 부분은 색상값과 채도를 떨어뜨려 밝은 부분을 더욱 두드러지게 만듭니다.
 
maximum : 이미지의 밝은 부분을 추가하여 밝은 이미지를 부각시킵니다.
 
minimum : 이미지의 어두운 부분을 추가하여 어두운 이미지를 부각시켜 줍니다.
 
offect : 수치를 입력하여 가로와 세로로 일정하게 이동시켜 줍니다.
